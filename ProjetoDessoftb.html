Projeto Final de Design de Software
Descrição do Projeto
Neste projeto, a classe foi dividida em grupos de 3 pessoas. Cada grupo deveria usar o conhecimento de programação adquirido no semestre, para fazer um jogo ou qualquer outra coisa que exigia certo nível de complexidade dos alunos.
Objetivos do projeto
O objetivo do projeto foi analisar o co-working do grupo, além de analisar se cada integrante estava dominando os conteúdos do semestre.Acesse o jogo!
